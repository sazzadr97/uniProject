Application can be found deployed on: https://smanageapp.herokuapp.com/

If you want to run in local host follow the steps (on windows):
1) clone this repo from git hub
2) in you command prompt enter the project directory
3) run $ pip install -r requirements.txt to insatll all requirements
4) run py manage.py makemigrations
5) run py manage.py migrate
6) run py manage.py runserver
7) go in the local url given and you will have access to all the application function
